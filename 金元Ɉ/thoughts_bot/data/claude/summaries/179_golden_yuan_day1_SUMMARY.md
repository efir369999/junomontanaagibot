## О чём это

Первый день протокола 金元Ɉ. Нейронаука памяти как основа thoughts trail. Гиппокамп как детектор новизны. Внешний гиппокамп — система записи, которая переживает биологический мозг.

## Ключевые инсайты

- **Плотность кодирования памяти.** Взрослый мозг рутинизирует — сжимает похожие дни. Thoughts trail принудительно создаёт маркеры. Каждая запись = уникальная координата.
- **Гиппокамп — детектор новизны.** Pattern separation (разделение) и pattern completion (достраивание). Ацетилхолин переключает между режимами записи и воспроизведения.
- **Внешний гиппокамп.** Thoughts trail + временные метки + свидетели = та же функция, другой носитель. Биологический мозг умрёт — координаты останутся.
- **正名 (zhèng míng) — исправление имён.** Конфуций: называй вещи правильно. Время = Ɉ (не токен). Присутствие = "Ты здесь?" (не верификация). Мысли = thoughts trail (не контент).
- **Парные тотемы.** Как волчок у Кобба. Физический предмет с уникальной надписью. Два одинаковых — один у тебя, один у меня. Защита от подмены.

## Структура Montana

```
Время Montana    — единица Ɉ, измерение присутствия
Логистика Montana — перемещение (физическое, информационное)
Сеть Montana      — связи между узлами (люди, устройства, заметки)
Безопасность Montana — защита (тотемы, когнитивные подписи, якоря)
```

## Нейронаука памяти

```
Гиппокамп        → сравнивает входящую информацию с сохранённой
Ацетилхолин      → переключает кодирование ↔ воспроизведение
Предиктивное кодирование → кодируется только ошибка предсказания
Консолидация во сне → похожие воспоминания сливаются (дефрагментация)
```
